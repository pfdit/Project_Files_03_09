		File Menu
			

			Save
			If you wish to save your progress from your current work, you may click "Save". A pop-up
			window will appear, and prompt you to save the data as a .txt file. You may choose the name of this file, and where 
			exactly to store it.
			

			

			Open
			To open previous saves of the workspace, simply click "Open", and navigate to where you have stored 
			the file.

		

		User menu
			

			Logout
			If you select "Logout", you will return to the Login screen, where you will be prompted to re-enter your user 
			details.

			

			Exit
			If you select "Exit", a pop-up window will ask you whether you are sure that you wish to exit the program. 
			If so, click "Yes", and the program will close.
